# Loan Document Analyzer

## Overview
An AI-powered Loan Document Analyzer that extracts key financial details from loan agreements using NLP and LLMs.

## Features
- Upload PDF
- Extract loan information
- Detect hidden charges
- RBI guideline comparison
- Risk score generation
- AI-generated summary

## Tech Stack
- Python
- FastAPI
- LangChain
- FAISS
- Sentence Transformers
- React
- Render
